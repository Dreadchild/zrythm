Zrythm User Manual
==================

## Dependencies
- Sphinx doc <http://sphinx-doc.org/>
- sphinx-intl
- guile
- `guix build utils` guile module
- texlive (for PDF)

## Building
This is built using `ninja -C build bundle_manual`
